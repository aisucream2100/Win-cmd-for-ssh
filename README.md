インストーラーを設置
バージョンが1.0.1にアップ
「インストーラーファイル:win_cmd_for_ ssh-x64_1.0.1setup.zip」
------------------------------------
使い方：
ホスト名または、ipアドレスを入力して、ユーザー名、パスワードを入力してください

間違っている場合アプリが閉じます。
--------------------------------------------------------------------------------
Q%A
1.アプリが起動しない
このアプリは.Net 10を使っているため、それ以前のバージョンのランタイムだと、実行できません。
.net 10ランタイムをインストール（https://builds.dotnet.microsoft.com/dotnet/WindowsDesktop/10.0.0/windowsdesktop-runtime-10.0.0-win-x64.exe）してから、もう一度試して下さい。
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
2.入れない
windowsの場合、PINを設定していると、ログインできないことがあります。PINを削除して、もう一度ログインしてください。
